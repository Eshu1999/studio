# **App Name**: DocConnect

## Core Features:

- Doctor Profile Creation: Doctors can create profiles with their specializations, experience, and availability.
- Appointment Scheduling: Users can view doctor profiles and book appointments based on available time slots.
- Video Call Integration: Secure and private video call functionality for consultations between doctors and patients at the scheduled time.
- Automated Summarization: After a video call, an AI tool summarizes the key points discussed during the consultation. This information can then be saved directly to the database for the patient.
- Appointment Reminders: Automated reminders sent to both doctors and patients before scheduled appointments.
- Payment Integration: Enable secure online payments for consultations.

## Style Guidelines:

- Primary color: Soft blue (#7EC4CF) to evoke trust and calmness, reflecting the healthcare environment.
- Background color: Very light desaturated blue (#F0F8FF) to provide a clean, clinical feel.
- Accent color: Teal (#368C76), a contrasting hue from the primary blue, used for call-to-action buttons.
- Font pairing: 'Inter' (sans-serif) for both headings and body text, providing a modern and readable interface.
- Use clean and recognizable icons to represent different functionalities.
- A clean and intuitive layout that allows users to navigate easily.
- Subtle transitions and animations to enhance user experience.
